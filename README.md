# Introducción a la Programación & Metodología de la Programación

Asignaturas cursadas durante el Grado de Ingeniería Informática en la Universidad de Almería (UAL)

# Introducción a la Programación

Asignatura centrada en el aprendizaje de los fundamentos de la programación utilizando Java como lenguaje principal.

Durante las prácticas y ejercicios se trabajan conceptos como:

- Variables y tipos de datos
- Operadores y expresiones
- Entrada y salida de datos
- Estructuras condicionales
- Bucles e iteración
- Métodos y modularidad
- Arrays y matrices
- Diseño básico de algoritmos
- Resolución de problemas computacionales

---

# Metodología de la Programación

Asignatura orientada al desarrollo de programas más estructurados y mantenibles, profundizando en el uso de Java y en buenas prácticas de programación.

Entre los contenidos trabajados destacan:

- Programación orientada a objetos
- Clases y objetos
- Encapsulación y modularidad
- Herencia y polimorfismo
- Uso de colecciones y estructuras de datos
- Manejo de excepciones
- Ficheros y entrada/salida
- Diseño y organización de proyectos
- Reutilización y mantenimiento de código
